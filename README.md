# MAX Secretary — Review Materials

MAX Secretary — бот-секретарь для рабочих чатов в MAX, предназначенный для фиксации поручений, сроков, исполнителей, наблюдателей, комментариев, ответов исполнителей и приемки результата постановщиком.

Этот репозиторий содержит только материалы для ознакомления:
- концепция проекта;
- архитектура;
- roadmap;
- описание MVP и pilot release;
- интеграции MAX и Битрикс24;
- эксплуатационная и deployment-документация;
- known limitations.

Исходный код backend/webapp в этот review-репозиторий не включен.
