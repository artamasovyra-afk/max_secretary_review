## False Positives From Raw Grep

The raw command can match:

- `DEBUG` because it contains the substring `BUG`;
- `MAX_WEBHOOK_DEBUG_LOG`;
- Python bytecode and cache files;
- dependency files under `backend/.venv`;
- generated frontend dependencies under `node_modules`, if present.

These are not backlog items and should be ignored by future TODO inventory runs.

Recommended future command:

```bash
rg -n "\b(TODO|FIXME|XXX|BUG)\b" backend webapp docs scripts \
  -g '!backend/.venv/**' \
  -g '!**/__pycache__/**' \
  -g '!webapp/node_modules/**' \
  -g '!webapp/dist/**' \
  -g '!*.pyc'
```
