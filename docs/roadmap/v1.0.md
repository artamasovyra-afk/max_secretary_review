# v1.0 — Pilot Stable Release

Status: completed.

## Goal

Create a pilot-stable baseline that can be deployed, smoke-tested, backed up, and evaluated by pilot users.

## Backend Scope

- Tasks.
- Multiple assignees.
- Observers.
- Comments.
- File metadata.
- Assignee responses.
- Requester acceptance/rejection.
- Inbox summary.
- Reminders.
- MAX webhook adapter.
- MAX Bot API adapter in explicitly enabled mode.
- Bitrix24 manual sync.
- RBAC/auth preparation.

## WebApp Scope

- Dashboard.
- Tasks list.
- Task details.
- Comments, file metadata, responses and acceptance actions.
- Bitrix24 sync status in Task Details.
- Dev auth context preparation.

## Deployment Scope

- Production Docker Compose.
- VPS deployment documentation.
- Operator guide.
- Backup/restore scripts and documentation.
- Release smoke script.
- Offline bundle documentation and scripts.

## Release Artifacts

- Pilot deployment report.
- Production deployment checklist.
- Pilot acceptance scenarios.
- Known limitations document.
- Security checklist.
- Operator guide.
- Backup and restore guide.

## Not Included

- Full MAX WebApp authentication.
- Automatic Bitrix24 sync triggers.
- Two-way Bitrix24 sync.
- Bitrix24 import.
- Real file storage.
- Production SSO.
- Full observability stack.
- Full closed-contour certification.

## Outcome

`v1.0.0` is the pilot stable baseline. It is suitable for controlled pilot validation, not final production certification.
