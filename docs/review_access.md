# Review access

Этот репозиторий предназначен только для ознакомления с проектом MAX Secretary.

## Как дать доступ

GitHub:
Repository → Settings → Collaborators and teams → Add people

Рекомендуемая роль:
Read

## Что получает коллега

- документацию;
- roadmap;
- release reports;
- описание архитектуры;
- описание интеграций и ограничений.

## Что не входит

- backend source code;
- frontend source code;
- production .env;
- секреты;
- backup-файлы;
- Git history основного репозитория.

## Как отозвать доступ

Repository → Settings → Collaborators and teams → Remove access
